# CASDC Documentation

Welcome to the CASDC (Complete Active Directory Server Controller) documentation.

## Getting Started

CASDC is a complete Windows Server Active Directory replacement built as a single static binary for Linux systems.

## Features

- Complete Active Directory replacement
- Exchange Enterprise email functionality
- DNS and DHCP services
- Security Operations Center
- Web-based management interface
- Multi-node clustering support
- Comprehensive backup and restore

## Quick Start

1. Download the CASDC binary
2. Run as root: `sudo ./casdc`
3. Access the web interface at https://your-server/
4. Login with the generated admin credentials

For detailed installation instructions, see the [Installation Guide](installation.md).
